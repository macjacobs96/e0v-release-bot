SLIDE 3 — 功能亮点 & 邮件模板

✨ E0V 版本释放机器人 · 功能亮点

群聊协作 · 智能识别 · 一键发版 · 可追溯
🧠

智能模块识别

  • 两阶段检测: 先看人写的前缀 (去 URL)，再全文回退
  • 健康检测 > SDK > APK 优先级
  • MR vs Build 自动分类
  • 多模块合并提交自动拆分
  • 避免 URL 中 Domain 等关键词误匹配
🤝

群聊协作模式

  • 多人并行提交各模块链接
  • 文件拖拽上传自动分类
  • 进度实时可见 (✅/⬜ 状态)
  • 收件人动态增删管理
  • 重置后立即开始下一轮
📧

标准化邮件模板

  • 三段式结构: 版本链接 / RN / 测试报告
  • 模块自动汇总 (不发则标注)
  • 附件自动打包 (Base64 解码)
  • SMTP STARTTLS 安全发送
  • 发送后自动清空状态
