E0V 版本释放机器人 — 交互链路

群聊 @机器人 → 链接收集 → 文件上传 → 预览 → 确认发送邮件

Phase 1: 收集阶段

  • · @机器人 + URL → 自动识别模块
  • · 先看人写的前缀 (去掉 URL)
  • · 健康检测 > SDK > APK 优先级
  • · 文件自动分类 (RN/测试报告)

Phase 2: 确认阶段

  • · 「发版」→ 预览邮件内容
  • · 「预览」→ 完整邮件查看
  • · 管理收件人列表
  • · 「发送」→ 最终确认

Phase 3: 发送阶段

  • · 组装标准邮件模板
  • · Base64 解码附件
  • · SMTP STARTTLS 发送
  • · 发送后自动清空状态
